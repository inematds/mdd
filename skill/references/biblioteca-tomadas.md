# Biblioteca de tomadas de câmera

Escolha o movimento pela **função** do beat, não por estética solta. Varie entre
painéis vizinhos — dois push-in seguidos matam o ritmo.

| Tomada | Quando usar |
| --- | --- |
| **Push-in lento** | criar tensão ou dar importância a algo |
| **Crash zoom** | impacto, surpresa, revelação súbita |
| **Orbit shot** | mostrar objeto/personagem com energia e volume |
| **Top shot** | geografia, processo, organização do espaço |
| **Macro close-up** | detalhe premium, textura, acabamento |
| **POV** | imersão, colocar o espectador dentro da cena |
| **Whip pan** | transição rápida entre dois beats |
| **Dolly lateral** | movimento elegante, acompanhar sem cortar |
| **Rack focus** | revelar informação trocando o foco entre planos |
| **Freeze impact** | terminar com força, congelar o ápice |

## Como casar tomada com beat

- **Abertura** quer estabelecer → top shot, dolly lateral, push-in lento.
- **Tensão/expectativa** → push-in lento, rack focus.
- **Revelação/virada** → crash zoom, orbit, rack focus.
- **Detalhe de valor** → macro close-up.
- **Imersão** → POV.
- **Fechamento** → freeze impact, push-in final, orbit terminando no produto.

## Sequências de planos prontas (8 painéis)

Use como esqueleto e adapte ao assunto.

**IA / tecnologia**
1. monitor vazio → 2. mão digitando o prompt → 3. dados entrando →
4. rosto/imagem sendo gerado → 5. timeline de edição → 6. câmera virtual →
7. festival/laurels → 8. chamada final (reveal).

**Imóvel de luxo**
1. fachada ao pôr do sol → 2. travelling pela entrada → 3. sala ampla →
4. macro do acabamento → 5. cozinha iluminada → 6. vista da varanda →
7. família usando o espaço → 8. logo/CTA na edição (não no prompt).

**Produto**
1. problema visual → 2. produto na sombra → 3. close de textura →
4. uso em ação → 5. comparação antes/depois → 6. detalhe técnico →
7. pessoa satisfeita → 8. fechamento de venda (reveal).

**Evento**
1. chegada → 2. público → 3. bastidores → 4. momento alto →
5. reação → 6. detalhe de marca → 7. ápice em freeze → 8. convite (reveal).

**Ação / esporte**
1. preparação tensa → 2. gatilho do conflito → 3. esforço em close →
4. câmera orbital no movimento → 5. impacto em slow motion →
6. deformação/suor macro → 7. virada → 8. freeze impact final.

**História emocional**
1. close no rosto → 2. objeto simbólico → 3. ambiente silencioso →
4. memória/flashback → 5. ponto de tensão → 6. virada →
7. resolução no rosto → 8. plano aberto de fechamento.
